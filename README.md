# Tape
## Запуск
Для запуска необходим конфиг-файл формата txt со следующими полями
[]readDelay - время чтения
[]writeDelay - время записи
[]shiftDelay - время сдвига на одну ячейку
[]rewindDelay - время полной перемотки
[]memoryLimit - ограничение памяти

Также потребуется файл с числами. С помощью Generator можно сгенерировать числа в нужном количестве и сохранить в txt-файл.
